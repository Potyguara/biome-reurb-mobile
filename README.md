# BIOME REURB

Aplicativo Flutter/Dart para coleta offline-first de dados em projetos de Regularização Fundiária Urbana - REURB.

## Módulos previstos

- Autenticação
- Projetos REURB
- Dashboard do cadastrador
- Cadastro físico
- Cadastro socioeconômico
- Selagem dos imóveis
- Mapa georreferenciado
- Documentos e fotos
- Sincronização offline/online
- Portal/consulta do beneficiário

## Comandos iniciais

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```
